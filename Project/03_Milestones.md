# 4. Completion Criteria

Bir milestone tamamlanmış sayılabilmesi için:

- Tüm görevler tamamlanmış olmalıdır.
- İlgili dokümantasyon güncellenmiş olmalıdır.
- Kod incelemesi (Code Review) tamamlanmalıdır.
- Testler başarılı olmalıdır.
- Kritik hata bulunmamalıdır.
- Git deposunda ilgili sürüm etiketi (tag) oluşturulmalıdır.
